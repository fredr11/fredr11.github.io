## NDIS Demand, Funding & Service Planning Analytics

**SQL • Power BI • DAX • Data Modeling • ETL**

Analyzed public NDIS and ABS datasets to identify participant growth trends, funding distribution patterns, and emerging service demand across Western Australia.

### Key Highlights
- Integrated 10 government datasets into a unified analytical model
- Built SQL transformation workflows for data cleansing and standardization
- Developed interactive Power BI dashboards with drill-down analysis
- Identified regions where NDIS participant growth exceeded population growth
- Generated insights to support service expansion and resource planning decisions
